## Lab Objectives
* To understand the concept of one-dimensional arrays in C, their declaration, initialization, and memory representation.
* To learn how to access, traverse, and modify array elements using loops and indexing.
* To understand how arrays store data in contiguous memory locations and how indexing works.
* To practice working with character arrays for storing and manipulating strings.
* To learn how to read and control user input using scanset (%[]) in scanf().
* To Understand the concept of negated scanset (^) to accept characters except specified ones.
* To develop the ability to process and analyze data stored in arrays efficiently using loops.
